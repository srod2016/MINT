# MINT

Money Income Networth Tracker (“MINT”),  is a modern finance tracker program. 

The program provides an easy and user-friendly experience for finance tracking that will help users better understand and optimize their finances.

The system includes different features such as:

- Expense tracking with categories and filters, budget setting and monitoring
- Basic and user-friendly finance reporting
- User authentication 
- Graphing and statistical representation of individual financial data
- Exporting capabilities
